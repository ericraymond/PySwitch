PySwitch Emulator


PySwitch is an open source firmware for MIDI foot controllers. It has been developed on PaintAudio's MIDI Captain series controllers, to control the Kemper Profiler Devices.

This website emulates PySwitch in your browser. See the documentation for details. You can:

  • Load the configuration from a connected controller running PySwitch (like a MIDI Captain) via MIDI into the emulator
  • Edit the configuration in a user friendly way with a Graphical User Interface
  • Save updated configuration files directly to the connected controller via MIDI

You have to allow MIDI connections for this website, your browser will ask you automatically for permissions.
Now, it would be just another configuration tool, if you could not do this, too:

  • Control any connected client device (like a Kemper Profiler) in realtime from your browser while editing the patches. The emulated device will control your Kemper just like the real one, even when no controller is connected.
  • Quickly open and test all examples delivered with PySwitch, connected to your real Kemper device or using the built-in virtual Kemper Profiler, all within the browser.
  • Scan for connected controllers and clients automatically
  • Edit the code in sync with the graphical editor: You can use your own implementations, they will not be destroyed by the code generator.

When you start up the application the first time, it will automatically scan all available MIDI ports for connected controllers. If one is found, it will automatically load its configuration.

After a configuration has been loaded, the application scans for available clients to control. The first one found (for example a Kemper Profiler) will be connected, or, if none has been found, a virtual Kemper Profiler is used. You can override this by clicking on the button at the bottom right and selecting a MIDI connection manually.
